# Workflow: Review

Before completing a major milestone, perform this cross-functional check:

## PM Review
- Does this feature align with the MVP scope in `PRD.md`?
- Is the "Approval" step explicitly implemented?

## UX/Identity Review
- Is Astra's tone stoic and minimal?
- Is the WhatsApp interaction as clean as possible?

## Engineering Review
- Is the code modular and typed?
- Are errors handled and logged?
- Is there any "magic" or overengineering?

## AI Review
- Is the prompt optimized for accuracy?
- Is context being handled efficiently (tokens/latency)?

## Security Review
- Are API keys protected?
- Is user data handled according to the "Minimal Persistence" rule?
